左手版本

本文所介绍的缺角层先法，很大程度上是一种解魔方的思路，而具体的操作方法可根据实际需要进行变换，从而构造出不同版本的还原方法。如果你惯用左手，或者是进行右手单拧，那么你一定希望有个左手版的还原方法。为了方便左手操作，我们可以将缺角工作区置于左前方。根据前面章节的原理，我们可以很方便地推导出左手版的缺角层先法。这里只给出公式，具体过程不再赘述。

1、换棱法

L' Ux L

F Ux F'

2、换角法

(D L D' L') Ux (L D L' D')

(L D L') Ux (L D' L')

(L² D L² D' L²) Ux (L² D L² D' L²)

3、转角法

(L D L' D') (L D L') Ux (L D' L') (D L D' L')

(L D' L') (D L D' L') Ux (L D L' D') (L D L')
